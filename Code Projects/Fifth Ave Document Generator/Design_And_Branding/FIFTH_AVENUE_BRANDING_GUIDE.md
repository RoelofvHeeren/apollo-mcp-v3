### 2.3 Vertical Text Treatment

One of the most distinctive Fifth Avenue design elements:

- **Rotation:** 90Â° counter-clockwise (reads bottom to top)
- **Position:** LEFT edge of page, centered vertically
- **Case:** UPPERCASE always
- **Weight:** Inter Light or Regular
- **Tracking:** Very wide letter-spacing
- **Color:** Black on light backgrounds, White on dark overlays

**Examples:**
- `CORPORATE PROFILE`
- `A NOTE FROM OUR LEADER`
- `KELOWNA` / `CALGARY` / `PHOENIX`
- `PEAK PERFORMANCE`

## 7. Spacing & Layout Rules

### 7.1 Page Setup

| Property | Value |
|----------|-------|
| **Orientation** | Portrait (vertical) |
| **Size** | Letter (8.5" Ã— 11") or A4 |
| **Margins** | 0.75" minimum all sides |
| **Grid** | Asymmetric 2-column common |

### 7.2 White Space Philosophy

> The brand uses **GENEROUS white space** as a luxury signifier.

| Element | Minimum Spacing |
|---------|-----------------|
| Page margins | 0.75 inches |
| Between sections | 40-60px |
| Between cards | 20-30px |
| Text line height | 1.5-1.7 |
| Paragraph spacing | 1.5Ã— font size |

### 7.3 The Signature Underline

Headlines and project names use a thin horizontal underline:
- **Weight:** 1-2px
- **Color:** Black or dark gray
- **Width:** Full width of text
- **Spacing below text:** 4-8px
