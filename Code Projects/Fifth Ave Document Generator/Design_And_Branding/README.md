## How to Use This Folder

1. **Read `FIFTH_AVENUE_BRANDING_GUIDE.md`** — This is the master reference with:
   - Logo usage rules
   - Typography hierarchy (Inter font)
   - Color palette
   - Layout templates with specifications
   - Spacing guidelines
   - Image treatment rules

2. **Use logos from `/logos/`** — Select the right variant:
   - Black logos for light backgrounds
   - White logos for dark backgrounds
   - Properties vs Homes depending on document type

3. **Reference `/corporate_profile_reference/`** — These are real page examples from the Fifth Avenue Corporate Profile showing exact layouts

4. **Install fonts from `/fonts/`** — Inter is the primary brand font

---

## Key Design Rules

| Rule | Value |
|------|-------|
| **Primary Font** | Inter (Light, Regular, Medium, SemiBold) |
| **Page Orientation** | Portrait |
| **Image Corners** | Sharp/rectangular (NO rounded corners) |
| **Logo Usage** | Sparingly — cover + back pages only |
| **White Space** | Generous — luxury aesthetic |
| **Vertical Text** | Section titles rotated 90° on left margin |

---

## Image Generation Workflow

When generating documents that need atmospheric images (NOT property renders):

1. Propose image locations and prompts for approval
2. Wait for user green light
3. Generate and incorporate

**Never generate:** Actual property renders, building designs, interiors (these must come from architects)
